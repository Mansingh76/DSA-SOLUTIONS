191. Number of 1 Bits (Hamming Weight)

Explanation:
n & (n-1) removes the rightmost 1-bit â†’ count total 1s efficiently.


class Solution {
    public int hammingWeight(int n) {
        int count = 0;
        while (n != 0) {
            n &= (n - 1); // remove last set bit
            count++;
        }
        return count;
    }
}
