371. Sum of Two Integers (without + or -)

Explanation:
Use bitwise XOR for addition without carry, AND + shift for carry.

class Solution {
    public int getSum(int a, int b) {
        while (b != 0) {
            int carry = a & b;
            a = a ^ b;          // sum without carry
            b = carry << 1;      // carry shifted left
        }
        return a;
    }
}
